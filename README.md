# Requirements
1. Ruby 2.7
2. Bundler
3. Rails
4. Yarn

# Env File
Please adjust the .env to reflect your db credentials.

# Automatic installation
`bash install.sh`

# Manual installation
1. `check your own dependencies`
2. `bundler install`
3. `rails webpacker:install`
4. `rails db:setup`
5. `rails db:seed`
